# AP World History: Modern

Welcome to the AP World History: Modern intelligent textbook project.

This site contains course notes, a detailed course description, and supporting
learning materials for a world history curriculum aligned with the College Board
AP World History: Modern framework (c. 1200 CE to the present).

- **Course Description** — full scope, topics, and Bloom's Taxonomy learning outcomes
- **World History Notes** — reference notes on frameworks and standards
- **Learning Graph** — concept maps and assessment reports
